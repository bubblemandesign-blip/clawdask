FAQ

Why does Windows show a security warning?

ClawDesk is not yet code-signed with an Extended Validation certificate. Click "More info" → "Run anyway" to proceed. SHA256 checksums are published with every release for verification.

What is OpenClaw?

OpenClaw is an open-source personal AI assistant (MIT license, 1,000+ contributors). ClawDesk wraps it into a desktop app so you don't need a terminal.

Is my data private?

Yes. OpenClaw runs locally. Your API keys and conversations stay on your machine. ClawDesk has no servers — nothing is sent to us.

Do I need Node.js?

No. ClawDesk now comes fully bundled with its own runtime. You don't need to install Node.js, Git, or anything else to use it.

Can I use self-hosted models?

Yes. Select "Custom" as your provider and enter your endpoint URL (e.g., Ollama, vLLM, or any OpenAI-compatible API).

Does ClawDesk start with my computer?

Optional. Right-click the tray icon → enable "Start with Windows". ClawDesk will boot silently and keep your AI ready in the background.
